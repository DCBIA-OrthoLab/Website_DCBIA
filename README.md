# DCBIA Website

A modern and responsive website to present the DCBIA working group.

## Description

Official website of the Digital Craniofacial Biology and Imaging Analysis (DCBIA) group presenting the team, research projects and publications.

## Installation

1. Clone the project
```bash
git clone https://github.com/DCBIA-OrthoLab/Website_DCBIA.git
```

2. Open `index.html` in your web browser

## Project Structure

```
Website_DCBIA/
├── index.html          # Homepage
├── team.html           # Team page
├── projects.html       # Projects page
├── styles/             # CSS files
├── scripts/            # JavaScript files
├── images/             # Images and resources
└── pdf/                # PDF documents
```

- HTML5
- CSS3

- Responsive design
